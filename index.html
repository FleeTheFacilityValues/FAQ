FAQ

Last Updated: August 12th

Who is mainly in charge of the value list?
The current value list holder and the person mainly in charge is Rox.

How are values determined?
Values are determined by a group of experienced traders, usually called Value List Helpers or Helpers for short. The main methods of determining a set's value include: observing frequent offers for that set, analyzing how often the set is being sought after, conducting community votes, gathering community suggestions, and other evaluation methods.

Why do we only assign values to sets and not individual items?
We do not assign values to individual items from a set, commonly referred to as “brokens”, because there is no reliable or consistent way to determine their values. The range of offers for brokens varies too widely, making it impossible to establish accurate and consistent values.

How often does the value list update? Is there a specific schedule?
There is no specific schedule for updates. The value list is updated after reviewing all suggestions, conducting votes, and determining which changes will be implemented based on the results. In other words, updates are released randomly. However, they typically occur every 2 to 7 days.

Will there be official conversion rates to other games?
No, there will not be any conversion rates to other games.

How many List Helpers are there?
The team currently consists of 6 members, including the Value List Holder.

Is it possible to join the value list team?
Yes, it is possible to become a member of the team through applications that we occasionally open in the official AW Apps Discord server.

When do applications for Value List Helper open?
There is no set schedule for applications. They are opened only when we are looking for new members. If applications are currently open, you will find that information in the pinned messages within the AW Apps Discord server, specifically in the trading channel.

Why are we reworking values?
To completely replace hunter values with a better & more stabilized value reference point, since hunters were not receiving a rise despite them getting more than their own value, so we look forward to improve the overall values of all the sets in game.

What is the new reference point?
A legendary set from Series 1 crate (any hammer + any gem can count as a set) that will be worth 1 value in the new system.

Why exactly this reference point and not something else?
Series 1 legendaries can't rise because they're always going to be obtainable, and can't drop any lower than we're valuing them already, because it's simply not worth spending credits on them considering the years they've been out for.


Important details:
You might NOT always be able to trade a 4 value set for 4 crate legendary sets and the other way around. Those will pretty much only work as a value reference point and not be relevant in usual trading. We are NOT encouraging you to buy any items from permanent crates for the purpose of trading them.
